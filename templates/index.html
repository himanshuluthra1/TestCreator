How It Works

  1. Upload your two Excel files above.
  2. Select which column in File 1 maps to which column in File 2 for each comparison.
  3. Choose the data type and comparison operator for each criterion.
  4. Add as many comparison criteria as needed.
  5. Click Compare to download a ZIP containing:
      • matched.xlsx – rows from File 2 that matched File 1
      • unmatched.xlsx – rows from File 1 with no match in File 2
