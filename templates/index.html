How It Works

  1. Upload File 1 as the order file, then upload File 2 to File 8 as comparison files.
  2. Select which column in File 1 maps to which column in File 2 for each comparison.
  3. Choose the data type and comparison operator for each criterion.
  4. Add as many comparison criteria as needed.
  5. The app compares File 1 with File 2 first. Then it compares previous unmatched rows with File 3, then File 4, and so on until comparison files end.
  6. Click Compare to download a ZIP containing:
      • matched.xlsx – combined matched rows from all comparison files
      • unmatched.xlsx – rows from File 1 still unmatched after all comparisons
